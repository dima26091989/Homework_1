HTML,CSS и JS

HTML

Html -стандартизированный язык разметки документов во всмирной паутине.Большинство веб-страниц содержат описание разметки на языке HTML.Язык HTML интерпретируется браузерами;полученный в результате интерпретации форматированный текст отображается на экране компьютера.

CSS

Формальный язык описания внешнего вида документа,написанного с использованием языка разметки.Приемущественно используется как средство описания,оформления вншнего вида веб-страниц,написанных с помощью языков разметки HTML и XHTML.

JS

JavaScript-мультипарадигменный язык программирования.Поддерживает объектно-ориентированный,императивный и функциональный стили.Является реализацией языка ECMAScript.
